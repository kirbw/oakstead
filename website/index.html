Features

Everything a small school needs

No bloat, no subscriptions. Families, grades, reports, and backups — all in one responsive interface that works on any device.

📚

Gradebook

Fast spreadsheet-style grade entry with custom weights, letter-grade scales, and automatic class and student averages.

👨‍👩‍👧‍👦

Families & Students

Enroll families and students, track grade placement, and manage classroom assignments across school years.

🏫

Classrooms & Teachers

Organize teachers and classrooms with admin, principal, teacher, and parent accounts, each with session-secure logins.

📅

School-Year Switcher

View and compare current and prior year records without losing a thing. History stays intact as years roll over.

📊

Reports

Built-in reports for class averages, individual student subjects, and birthday schedules — ready to print or export.

💾

SQLite & Backups

Your data lives in a single local SQLite file. Create, download, and restore backups right from the admin panel.

📱

Responsive on Any Device

Works great on iPhones, iPads, and desktops — no app to install, just open the browser.

🔄

Self-Updating

Pull stable or pre-release updates directly from the admin panel without touching the command line.

Installation

Explore first, then host it yourself

Oakstead currently installs from source on Windows or Linux. No separate database server is required.

Install from source

Run Oakstead on Windows or Linux with Node.js 22.13 or newer. A tested Windows installer is not currently published.

Review release notes and source changes as Oakstead grows. Windows packaging remains under development and is not release-ready.

Designed for trusted networks. Oakstead stores sensitive school records. Use a local network or VPN, and protect any remote access with HTTPS, request filtering, and appropriate access controls.
